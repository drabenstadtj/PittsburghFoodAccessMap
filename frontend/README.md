# Pittsburgh Food Access Map - Frontend

React frontend for the food resource map.

## Quick Start
```bash
# Setup
npm install

# Run
npm start
```

Frontend runs at 'http://localhost:3000'

## What's Working Now
- Basic react app that runs and displays a map with some plotted points
